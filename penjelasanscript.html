Penjelasan script

  • head

    • Ini adalah elemen pertama yang menandai awal bagian utama dokumen HTML. Ini termasuk informasi tentang dokumen HTML, seperti informasi meta, link ke file eksternal (CSS, JavaScript), nama halaman, dll.

  • meta charset="UTF-8"

    • Ini adalah tag meta yang menentukan karakter yang digunakan dalam dokumen HTML. Dalam hal ini, \"UTF-8" menunjukkan pengkodean karakter yang akan digunakan Unicode. Hal ini penting untuk memastikan font yang digunakan pada halaman web dapat ditampilkan di semua perangkat dan browser.

  • meta name="viewport" content="width=device-width, initial-scale=1.0"

    • Ini adalah tag meta yang menginstruksikan browser cara menampilkan konten halaman web di perangkat. terpisah. Dalam hal ini, width=device-width memberitahu browser untuk menyetel lebar lamannya berdasarkan lebar perangkat, dan skala awal=1.0 menyetel tingkat zoom awal laman web saat pertama kali dimuat di perangkat ini.

  • title Halaman Heading title

    • Inilah yang menentukan judul halaman web. Nama ini akan ditampilkan di bilah judul browser dan akan muncul sebagai nama halaman ketika halaman tersebut ditandai sebagai favorit atau dibagikan kepada publik.
